Ollama setup

1. Install Ollama if you haven't already.

2. Pull the embedding model:
ollama pull nomic-embed-text

3. Start Ollama with extension access enabled:
OLLAMA_ORIGINS="*" ollama serve

Built-in model

The first time a tab is moved, the extension will download all-MiniLM-L6-v2 (~25 MB) from Hugging Face and cache it in your browser. After that, it works fully offline. You can track download progress in the popup.
Domain-only matching

Tabs are grouped by their hostname (e.g. all github.com tabs together). No AI, no downloads, no external servers. You can always switch to a smarter backend later in the popup.
